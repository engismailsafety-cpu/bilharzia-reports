# نظام التقارير الأسبوعية للبلهارسيا والفاشيولا

نظام إلكتروني متكامل لإدارة التقارير الأسبوعية للإصابة بالبلهارسيا والفاشيولا بوحدة الجزيرة الخضراء الصحية.

## 🚀 المميزات

- إدخال البيانات بسهولة
- حساب الإجماليات تلقائياً
- رقم الأسبوع الدولي مع إمكانية التعديل
- حفظ التقارير في المتصفح
- تصدير PDF جاهز للطباعة
- إحصائيات شهرية

## 📝 كيفية الاستخدام

1. اختر تاريخ نهاية الأسبوع
2. أدخل البيانات في الجدول
3. احفظ التقرير أو استخرج PDF
